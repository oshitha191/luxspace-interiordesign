# Luxury Interior Design Website

## Overview
A premium, responsive static website for an Interior Design Company built using HTML, CSS, and JavaScript.

## Tech Stack
- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll)

## Features
- Responsive Design
- Hero Section
- About Company
- Featured Products
- Services
- Our Process
- Statistics
- Testimonials
- Product Catalog
- Search UI
- Category Filter UI
- Smooth Animations
- Hover Effects

## Libraries Used
- AOS Animation Library
- Google Fonts



## Setup Instructions

1. Download the repository.
2. Open the project folder in VS Code.
3. Install the Live Server extension.
4. Right-click `index
.html`.
5. Select Open with Live Server.
